An **interactive disaster simulation website** that demonstrates how different natural disasters behave using **animations, visual indicators, and interactive elements**.

The project is designed as a **single-page application (`index.html`)** using **HTML, CSS, and JavaScript**.

---

# 🌋 Project Overview

The **Natural Disaster Simulator** allows users to explore how different natural disasters occur and what their impacts are.

Users can simulate disasters like:

🌋 **Volcano Eruption**
🌊 **Tsunami Wave**
🌪 **Hurricane Storm**
🌍 **Disaster Zones on World Map**

Each disaster includes **animations and a severity meter** to demonstrate intensity.

---

# 🚀 Features

✨ **Volcano Eruption Animation**
Lava erupts from a volcano using CSS animation.

🌊 **Moving Tsunami Wave**
Water wave animation simulating tsunami movement.

🌪 **Spinning Hurricane**
Rotating storm animation representing hurricanes.

🌍 **Interactive World Map**
Clickable markers showing disaster-prone zones.

🚨 **Disaster Severity Meter**
Visual indicator showing the intensity of disasters.

---

# 🖥 Demo Simulation Panel

The interface includes buttons for each disaster.

Example controls:

🌋 Volcano
🌊 Tsunami
🌪 Hurricane
🌍 World Map

Selecting a disaster updates the **simulation area dynamically**.
